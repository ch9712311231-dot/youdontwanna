## 3. 표지 최종 사양

여러 라운드의 피드백(레퍼런스 이미지 2종 반영 → 전문 디자이너 관점 6가지 지시 → 로고 위치 재변경)을 거쳐 아래로 최종 수렴함:

- **배경**: 연한 하늘색 그라데이션 `linear-gradient(168deg, #F2FBFD 0%, #DEF4FA 45%, #BFE8F2 100%)` + 흰색/남색 톤 원형 데코 3개.
- **좌측 상단**: "1위 딱지" 리본 — 남색 그라데이션, 하단 제비꼬리 컷, "교원임용 / 1위 / 해커스". 바로 아래 3줄 근거문구(수동 줄바꿈): "2021 대한민국 No.1 대상 교원임용 교육 / (온·오프라인 교원임용) 부문 1위 / (한국미디어마케팅그룹)".
- **우측 상단**: 현재 비어 있음(로고를 하단으로 옮겼기 때문).
- **eyebrow**: "시작하는 중등 임용고시 수험생을 위한" — 흰 pill 배지, 13.5pt.
- **메인 타이틀**: 66pt, 굵게, 2줄 색 분리 — `해커스임용`(진한 남색) / `합격 로드맵`(브랜드 시안색, 로고 색과 동일).
- **서브타이틀 박스**: "중등 임용고시 대비" — 남색 배경, 흰 글씨 15pt, radius 2.2mm(덜 둥글게 조정됨).
- **설명 박스**: 2단 구성 — 메인 문구(13pt, 굵게) + 구분선 아래 서브 문구(10.5pt): "해커스임용 스타 강사진이 제시하는 과목별 합격 전략 / 중등 임용고시 최신 정보 수록".
- **하단(footer)**: 좌측 로고(흰 박스 없이 원본, 9mm 높이) / 우측 `teacher.Hackers.com`.
- **삭제된 요소**: "합격으로 가는 길" SVG 로드맵 그래픽(시작→경로→깃발)은 최종적으로 제거하고 타이포 확대로 대체.

관련 코드: `src/book.css`의 "표지" 섹션(`.cover` 전부) + `src/book.html`의 `<section class="page nopad cover">`.

## 5. 프로젝트 파일 구조

```
C:\임용AtoZ\
├─ src\book.html          ← 전체 페이지 원본(모든 <section class="page">가 한 페이지)
├─ src\book.css            ← 디자인 시스템 전체(색상·타이포·컴포넌트 전부 여기)
├─ render.js                ← book.html → PDF 변환 스크립트 (puppeteer-core + Edge)
├─ build_web.js             ← book.html → 자기완결형 웹 미리보기 HTML 변환 스크립트
├─ assets\logo\             ← 해커스임용 로고 PNG (color/black/white)
├─ assets\fonts\             ← Pretendard 원본 폰트(전체 글립셋, PDF용)
├─ assets\fonts_subset\      ← Pretendard 서브셋 폰트(웹뷰어 전용, ~240KB)
├─ _source\pdf_tool.py       ← 레퍼런스 PDF를 PNG로 렌더링/텍스트 추출하는 유틸(PyMuPDF)
├─ _source\excel_dump.txt    ← 모집인원 엑셀 원본 덤프
├─ output\임용합격로드맵_draft.pdf   ← 최종 PDF 산출물
├─ output\임용합격로드맵_web.html   ← 더블클릭으로 여는 독립형 웹 뷰어(폰트·이미지 base64 내장)
└─ output\_artifact_fragment.html ← Artifact 게시 전용(래퍼 없는 조각)
```

## 6. 빌드 방법

```bash
cd C:\임용AtoZ
node render.js book.html <출력파일명>.pdf   # PDF 재생성 → output/ 폴더
node build_web.js                              # 웹 뷰어 재생성 → output/ 폴더
```

**book.html 또는 book.css를 수정하면 위 두 명령을 반드시 둘 다 다시 실행할 것** — 하나만 하면 다른 산출물이 구버전으로 남습니다.

- 판형: 트림 188×257mm + 도련 3mm = 작업 194×263mm.
- 색상: 브랜드 시안 `#00ACC8` 기준 팔레트(`--brand`, `--brand-deep`, `--brand-ink`, `--sky-50~300`), `book.css`의 `:root`에 정의.
- 폰트: Pretendard(로컬 woff2, 네트워크 불필요).
- 디자인 원칙: 모든 카드/배지는 둥근 모서리만 사용(각진 도형 금지).

## 7. book.css 주요 컴포넌트

- `.blk` / `.mh` / `.bd`: 기본 카드형 섹션(아이콘+제목 헤더 + 본문)
- `.credit`: 챕터 상단 저자 크레딧 배지
- `.type-grid` / `.tc`: 2열 카드 그리드
- `.qa-blk`: Q&A 카드
- `.era-list` / `.tag`: 가변폭 사각 태그 + 설명 리스트 (여러 글자 라벨은 반드시 이걸 사용 — 원형 배지 `.pn`은 숫자/한 글자 전용)
- `.two-col`: 표 2단 배치(행이 많을 때)
- `.ychart`, `.delta-grid`/`.delta-card`: 통계 차트/증감 카드
- `.instructor`: 강사 프로필 카드 — PART 2 템플릿

## 8. 알아둘 것 (버그·주의사항)

- 표 헤더(`<th>`)에 `.hl` 클래스를 쓰면 진한 배경과 겹쳐 글씨가 안 보이는 버그가 있었음 → th 강조는 밝은 색 인라인 스타일로.
- 레퍼런스 PDF는 `pdftotext`로 한글이 깨짐 → PyMuPDF(`_source/pdf_tool.py`)로 PNG 렌더링 후 시각적으로 읽어야 함.
- 웹뷰어 폰트는 현재 쓰인 문자만 추출한 서브셋 — PART 2 원고에서 서브셋에 없는 새 문자를 쓰면 웹뷰어에서만 글자가 깨질 수 있음(PDF는 안전). 문제 생기면 `fontTools.subset`으로 재서브셋 필요.
