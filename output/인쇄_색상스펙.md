# 해커스임용 합격로드맵 — 인쇄 사양 메모 (인쇄소 전달용)

## 1. 기본 사양
- 판형(트림): A5, 148mm × 210mm
- 도련: 사방 3mm → 작업(도련 포함) 크기 154mm × 216mm
- 제본: 중철(스테이플) 또는 무선제본 — 페이지 수 확정 후 인쇄소와 재확인 필요
- 총 페이지 수: 27p (표지 포함)
- 인쇄 방식: 2도인쇄 — 검정(K) + 브랜드 스팟 컬러 **PANTONE 3125 C** 1색

## 2. 전달 파일 (2026-07-20 갱신)
- `임용합격로드맵_인쇄용.pdf` — 도련(154×216mm) 기준 재단선(크롭마크)+슬러그라인 포함, **RGB** 원본
- `임용합격로드맵_인쇄용_CMYK.pdf` — 위 파일을 Ghostscript로 **DeviceCMYK 색공간 변환**한 파일(전 페이지 RGB 요소 없음, 확인 완료)
- 원본 디자인은 웹/HTML 기준 RGB로 제작되었으나, 이번에 브랜드 색을 **PANTONE 3125 C의 공식 디지털값(Coated)** 에 맞춰 보정했습니다. 아래 3절 표를 인쇄소 사전인쇄(prepress) 단계에서 참고해 주세요.

### ⚠️ 중요 — "CMYK 파일"과 "2도 스팟 인쇄"는 서로 다른 개념입니다
- `_CMYK.pdf`는 인쇄소 시스템이 RGB를 받지 못하는 경우를 대비한 **4원색 프로세스 변환본**입니다. Ghostscript 표준 변환 프로파일로 자동 변환한 값이라 실제 PANTONE 잉크 배합값과 정확히 일치하지 않습니다.
- **실제 2도(K+스팟) 옵셋 인쇄를 진행한다면, 인쇄소에는 "PANTONE 3125 C + 검정 2도 인쇄"라고 명확히 지정**하고, `_CMYK.pdf`는 색상 참고용/시스템 호환용으로만 쓰는 것을 권장합니다. CMYK 파일 그대로 4도(프로세스 컬러)로 인쇄하면 아래 스팟 컬러표와 실제 발색이 달라질 수 있고, 보통 2도 인쇄보다 단가도 높습니다.
- 인쇄소가 진짜 별색 분판(스팟 컬러 플레이트 2장: K 전용/스팟 전용)을 요구하는 경우, 현재 파이프라인(브라우저 기반 PDF 출력)은 네임드 스팟 컬러 채널을 만들 수 없습니다 — 필요 시 알려주시면 K 단독/스팟 단독 2개의 그레이스케일 분판용 PDF를 별도로 제작해 드릴 수 있습니다.

## 3. 색상 사양 — PANTONE 3125 C 확정

### 브랜드 스팟 컬러 원색
| 항목 | 값 |
|---|---|
| PANTONE 번호 (Coated) | **3125 C** |
| 공식 디지털 HEX | `#00AEC7` |
| 공식 CMYK 참고값 | C100 M13 Y0 K22 |
| 근거 | 웹 디자인 색상 `#00ACC8`과 RGB 채널 오차 ±1~2 수준(육안 구분 불가)으로, 인쇄용 색상표(Pantone Color Bridge) 대조 시 가장 근접한 번호로 확인됨. **실물 컬러칩 대조는 인쇄 전 최종 권장.** |

### Plate A — 검정(K 100%), 본문·텍스트용
| 변수 | HEX | 용도 | 근사 톤(K 잉크 농도) |
|---|---|---|---|
| `--ink` | #181818 | 본문 글자 | K 100% |
| `--ink-soft` | #4D4D4D | 보조 텍스트 | K 약 77% tint |
| `--muted` | #8C8C8C | 캡션·저작권 안내문 등 | K 약 50% tint |
| `--gray-50` | #F2F2F2 | 옅은 배경 | K 약 6% tint |
| `--gray-200` | #D6D6D6 | 옅은 테두리 | K 약 18% tint |

### Plate B — PANTONE 3125 C, 스팟 컬러 (전부 동일 1색의 tint 또는 K 오버프린트)
| 변수 | HEX | 실제 인쇄 배합 | 용도 |
|---|---|---|---|
| `--brand` | #00AEC7 | **PANTONE 3125 C 100%** | 표지 메인 타이틀, 표 헤더, 포인트 강조 |
| `--brand-deep` | #017A93 | PANTONE 3125 C 100% **+ K 약 28% 오버프린트** | 진한 강조(표 헤더 등) |
| `--brand-ink` | #0A4A5C | PANTONE 3125 C 100% **+ K 약 55% 오버프린트** | 표지 메인 타이틀 보조색·제목 강조 |
| `--line` | #D7ECEF | PANTONE 3125 C 약 26% tint | 구분선 |
| `--sky-50` | #F2FBFC | PANTONE 3125 C 약 5% tint | 카드·배경(가장 옅음) |
| `--sky-100` | #E3F6F8 | PANTONE 3125 C 약 12% tint | 카드·배경 |
| `--sky-200` | #CDEEF3 | PANTONE 3125 C 약 21% tint | 카드·배경 |
| `--sky-300` | #A9E1EA | PANTONE 3125 C 약 37% tint | 카드·배경(가장 진함) |

> tint %는 백색 용지 위 스팟 잉크 망점(halftone screen) 농도, 오버프린트 %는 스팟 잉크 100% 위에 얹는 K 잉크 망점 농도입니다. 두 값 모두 RGB 역산 근사치이므로 최종 톤은 인쇄소 교정쇄(프루프)로 확인 권장합니다.

## 4. 확인 요청 사항
1. ~~근사 CMYK/Pantone 값을 인쇄소 표준 컬러차트로 대조해 실제 인쇄용 스팟 컬러 확정~~ → **PANTONE 3125 C로 확정**(2026-07-20). 실물 Color Bridge 칩 대조만 최종 권장.
2. `--brand-deep`/`--brand-ink`는 K 오버프린트 방식으로 스펙 확정(3절 표 참고) — 인쇄소가 오버프린트 대신 별도 K 단독 tint로 처리하길 원하면 사전 협의 필요.
3. 최종 페이지 수 기준으로 중철/무선제본 중 인쇄소 권장 방식 재확인
4. 인쇄용 PDF의 재단선 위치·여백이 인쇄소 장비 규격에 맞는지 확인 (여백 10mm로 제작됨)
5. **2도 스팟 인쇄로 진행할지 CMYK 4도 프로세스 인쇄로 진행할지 인쇄소와 최종 확인** — 2절의 주의사항 참고. 답에 따라 전달해야 할 파일 종류가 달라집니다.
